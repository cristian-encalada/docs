# Introducción

La siguiente cheatsheet de Markdown se adapta de: https://guides.github.com/features/mastering-markdown/

# ¿Qué es Markdown?

Markdown es una forma de dar estilo al texto en la web. Tú controlas la presentación del documento; formatear palabras como negrita o cursiva, agregar imágenes y crear listas son solo algunas de las cosas que podemos hacer con Markdown. En su mayoría, Markdown es simplemente texto normal con algunos caracteres no alfabéticos añadidos, como `#` o `*`.

# Sintaxis

A continuación veremos de manera general la sintaxis de Markdown que puedes utilizar en cualquier lugar en GitHub.com o en tus propios archivos de texto.

## Encabezados

```
# Esta es una etiqueta h1

## Esta es una etiqueta h2

#### Esta es una etiqueta h4
```

# Esta es una etiqueta h1

## Esta es una etiqueta h2

#### Esta es una etiqueta h4

## Énfasis

```
_Este texto estará en cursiva_

**Este texto estará en negrita**

_Puedes **combinar** ambos_
```

_Este texto estará en cursiva_

**Este texto estará en negrita**

## Listas

### No ordenada

```
- Elemento 1
- Elemento 2
  - Elemento 2a
  - Elemento 2b
```

- Elemento 1
- Elemento 2
  - Elemento 2a
  - Elemento 2b

### Ordenada

```
1. Elemento 1
1. Elemento 2
1. Elemento 3
   1. Elemento 3a
   1. Elemento 3b
```

1. Elemento 1
1. Elemento 2
1. Elemento 3
   1. Elemento 3a
   1. Elemento 3b

## Tablas

Puedes crear tablas ensamblando una lista de palabras y dividiéndolas con guiones `-` (para la primera fila), y luego separando cada columna con una barra `|`:

```
| Primer Encabezado               | Segundo Encabezado              |
| ------------------------------- | ------------------------------- |
| Contenido de la celda 1         | Contenido de la celda 2         |
| Contenido en la primera columna | Contenido en la segunda columna |
```

| Primer Encabezado               | Segundo Encabezado              |
| ------------------------------- | ------------------------------- |
| Contenido de la celda 1         | Contenido de la celda 2         |
| Contenido en la primera columna | Contenido en la segunda columna |

## Tachado

Cualquier palabra envuelta con dos virgulillas (como `~~esto~~`) aparecerá ~~tachada~~.
